# 틈서울 (TeumSeoul)

> 갑자기 생긴 틈을, 지금 즐길 수 있는 서울 경험으로.

틈서울은 사용자의 **남은 시간·위치·분위기·동행자**를 바탕으로 지금 방문할 수 있는 서울의 장소와 코스를 추천하는 통합 탐색 서비스입니다. 서울 공공데이터와 익명 현장 정보를 연결해 장소 탐색부터 지도, 상세 정보, 다음 행동까지 한 흐름으로 제공합니다.

🌐 **배포 URL:** [서비스 바로가기](https://teumseoul.netlify.app)

## 주요 기능

- 상황 기반 장소 및 코스 추천
- 서울 공공장소 6,500여 건의 지도 탐색과 상세 정보 제공
- 실제 장소 데이터에 근거한 AI 챗봇
- 혼잡도, 방문 팁 등을 공유하는 익명 커뮤니티
- 운영 정보 누락 및 추천 한계를 숨기지 않는 명시적 안내

## 신뢰할 수 있는 추천

틈서울은 AI가 장소를 자유롭게 만들어 내지 않도록 **DB 검색 → 규칙 검증 → AI 선택 → 서버 재검증** 순서로 추천합니다. 시간·거리·장소 ID를 서버에서 확인하며, AI 요청이 실패하면 규칙 기반 추천으로 전환합니다.

## 기술 스택

- **Frontend:** Vue 3, Vite, Leaflet
- **Backend:** FastAPI, SQLAlchemy, SQLite
- **AI:** OpenAI API
- **Deploy:** Netlify, Render


- API 문서: `backend/.docs`
